# MarisMasses
Hymn Lyric management app for Stella Maris Church Miramar

# MassCast

Offline-first Catholic hymn presentation system.

## Requirements

Python 3.13+

## Installation

```bash
python -m venv .venv
```

Windows

```bash
.venv\Scripts\activate
```

Linux / Raspberry Pi

```bash
source .venv/bin/activate
```

Install packages

```bash
pip install -r requirements.txt
```

Run

```bash
python run.py
```

Open

http://localhost:8000
